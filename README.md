# 小红书收藏归档 · 代号鸢

把小红书收藏夹里的帖子链接发给助手，自动整理成**可翻页、带分区**的图文归档文档。

- 每篇一页，全文直接可见，不用点开原帖
- 顶部按「分区」筛选，目录按分区分组
- 支持导出 PDF / EPUB（`index.pdf` / `index.epub`）

## 在线查看（永久入口）
开启 GitHub Pages 后，本仓库网址即为永久地址：

`https://bellali2336.github.io/xhs-archive/`

本仓库为**完全自包含**：`index.html` 与 `images/` 图片全部在仓库内，不依赖任何外部托管，离线也能看。

## 如何更新
每天把新链接发给助手并说明归入哪个分区，助手会重新生成文件；
你只需在本目录双击 `sync_github.bat`（首次会弹浏览器登录 GitHub，无需 token），
网址不变，内容自动更新。
