# 小红书收藏归档 · 代号鸢

把小红书收藏夹里的帖子链接发给助手，自动整理成**可翻页、带分区**的图文归档文档。

- 每篇一页，全文直接可见，不用点开原帖
- 顶部按「分区」筛选，目录按分区分组
- 支持导出 PDF / EPUB

## 在线查看（永久入口）
开启 GitHub Pages 后，本仓库网址即为永久地址：

`https://bellali2336.github.io/xhs-archive/`

页面内嵌了实时归档（含图片），打开就是最新合集，地址栏始终显示本 GitHub 网址。

## 说明
- `index.html` 是一个极小的内嵌页，加载实时归档内容与图片。
- 如需把完整网页与图片都打包进本仓库（完全自包含、不依赖外部托管），在本机运行 `sync_github.bat` 即可。

## 如何更新
每天把新链接发给助手并说明归入哪个分区，助手会重新生成并推送，网址不变。
